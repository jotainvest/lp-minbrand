Se não tiver recebendo seu código, Veja abaixo como receber!

Alguns contatos de WhatsApp utilizam certificados digitais criptografados conectados ao número de acesso para garantir que o mesmo número que gerar o código é o mesmo número que irá validâ-lo, então por esse motivo há uma certa dificuldade para receber o código do QR Code de acesso de contatos com esse sistema.

Caso esteja com esse problema, poderá ser necessário adicionar um certificado DNFI ao número que irá receber o código também.
